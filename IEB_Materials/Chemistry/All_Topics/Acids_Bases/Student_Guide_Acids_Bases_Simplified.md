## **Chapter 1: What Are Acids and Bases?**

### **Simple Definitions:**

#### **Acids:**
- **Taste sour** (lemon, vinegar)
- **Turn blue litmus red**
- **pH less than 7**
- **React with metals** to produce hydrogen gas
- **Examples:** HCl, H‚āāSO‚āĄ, CH‚āÉCOOH, lemon juice

#### **Bases:**
- **Taste bitter** (baking soda, soap)
- **Turn red litmus blue**
- **pH greater than 7**
- **Feel slippery** (soapy)
- **Examples:** NaOH, NH‚āÉ, baking soda, soap

### **Memory Trick:**

## **Chapter 2: Three Acid-Base Theories**

### **1. Arrhenius Theory (Simplest):**
- **Acid:** Produces H‚Āļ ions in water
- **Base:** Produces OH‚ĀĽ ions in water
- **Example:** HCl ‚Üí H‚Āļ + Cl‚ĀĽ (acid), NaOH ‚Üí Na‚Āļ + OH‚ĀĽ (base)
- **Limitation:** Only works in water

### **2. Br√łnsted-Lowry Theory (Most Useful):**
- **Acid:** Proton (H‚Āļ) DONOR
- **Base:** Proton (H‚Āļ) ACCEPTOR
- **Works in any solvent**
- **Introduces conjugate pairs**

### **3. Conjugate Acid-Base Pairs:**
- **Acid loses H‚Āļ ‚Üí becomes conjugate base**
- **Base gains H‚Āļ ‚Üí becomes conjugate acid**
- **Example:** HCl/Cl‚ĀĽ (acid/conjugate base), NH‚āÉ/NH‚āĄ‚Āļ (base/conjugate acid)

### **Memory Trick:**
```
Arrhenius: H‚Āļ and OH‚ĀĽ in water only
Br√łnsted-Lowry: Proton donor/acceptor anywhere
```

## **Chapter 3: The pH Scale**

### **What is pH?**
- **pH = -log‚āĀ‚āÄ[H‚Āļ]**
- **Measures acidity/basicity**
- **Scale:** 0 (very acidic) to 14 (very basic), 7 (neutral)

### **pH Values of Common Substances:**
```
0-1: Battery acid, stomach acid
2-3: Lemon juice, vinegar
4-5: Tomato juice, black coffee
6: Milk, urine
7: Pure water (neutral)
8: Sea water, eggs
9-10: Baking soda, antacids
11-12: Ammonia, soap
13-14: Drain cleaner, NaOH
```

### **Important Relationships:**
- **pH + pOH = 14** (at 25¬įC)
- **Each pH unit** = 10√ó change in [H‚Āļ]
- **pH 3** has 10√ó more H‚Āļ than **pH 4**
- **pH 2** has 100√ó more H‚Āļ than **pH 4**

### **Calculating pH:**
1. **For strong acids:** pH = -log[H‚Āļ] (direct)
2. **For strong bases:** pOH = -log[OH‚ĀĽ], then pH = 14 - pOH
3. **For weak acids/bases:** Need Ka/Kb (see Chapter 5)

## **Chapter 4: Strong vs Weak Acids/Bases**

### **Strong Acids/Bases:**
- **Completely dissociate** in water
- **Good conductors** of electricity
- **React vigorously**
- **Examples:**
  - **Strong acids:** HCl, H‚āāSO‚āĄ, HNO‚āÉ, HClO‚āĄ
  - **Strong bases:** NaOH, KOH, Ca(OH)‚āā

### **Weak Acids/Bases:**
- **Partially dissociate** in water
- **Poor conductors** of electricity
- **React slowly**
- **Examples:**
  - **Weak acids:** CH‚āÉCOOH (vinegar), H‚āāCO‚āÉ, citric acid
  - **Weak bases:** NH‚āÉ (ammonia), baking soda

### **Memory Trick:**
```
STRONG = Separates Totally, Reacts Obviously, Never Goes back
WEAK = Won't Entirely Associate, Keeps some together
```

## **Chapter 5: Ka, Kb, and Kw**

### **Acid Dissociation Constant (Ka):**
- **Measures acid strength**
- **Ka = [H‚Āļ][A‚ĀĽ]/[HA]** for HA ‚áĆ H‚Āļ + A‚ĀĽ
- **Large Ka** = strong acid
- **Small Ka** = weak acid
- **pKa = -logKa** (like pH)

### **Base Dissociation Constant (Kb):**
- **Measures base strength**
- **Kb = [BH‚Āļ][OH‚ĀĽ]/[B]** for B + H‚āāO ‚áĆ BH‚Āļ + OH‚ĀĽ
- **Large Kb** = strong base
- **Small Kb** = weak base
- **pKb = -logKb**

### **Water Ionization Constant (Kw):**
- **Kw = [H‚Āļ][OH‚ĀĽ] = 1.0 √ó 10‚ĀĽ¬Ļ‚Āī** at 25¬įC
- **Always true** in aqueous solutions
- **Neutral water:** [H‚Āļ] = [OH‚ĀĽ] = 1.0 √ó 10‚ĀĽ‚Ā∑ M
- **Relationship:** Ka √ó Kb = Kw for conjugate pair

## **Chapter 6: Titrations**

### **What is a Titration?**
- **Method to find concentration** of unknown solution
- **Add known solution** from burette until reaction complete
- **Endpoint:** When indicator changes color
- **Equivalence point:** When moles acid = moles base

### **Titration Setup:**
```
Burette (known concentration)
   ‚Üď
   ‚Üí Titrant added to ‚Üí
   ‚Üď
Erlenmeyer flask (unknown + indicator)
```

### **Titration Calculations:**
**Formula:** M‚āĀV‚āĀ = M‚āāV‚āā (for 1:1 ratio)
- **M‚āĀ** = concentration of known
- **V‚āĀ** = volume of known used
- **M‚āā** = concentration of unknown
- **V‚āā** = volume of unknown

**Example:** 25.0 mL HCl titrated with 0.1 M NaOH. 30.0 mL NaOH used.
M‚āĀV‚āĀ = M‚āāV‚āā ‚Üí 0.1 √ó 30.0 = M‚āā √ó 25.0 ‚Üí M‚āā = 3.0/25.0 = 0.12 M HCl

### **Common Indicators:**
- **Phenolphthalein:** Colorless (acid) ‚Üí Pink (base), pH 8.2-10.0
- **Methyl orange:** Red (acid) ‚Üí Yellow (base), pH 3.1-4.4
- **Universal indicator:** Shows full pH range with rainbow colors

## **Chapter 7: Buffer Solutions**

### **What is a Buffer?**
- **Resists pH change** when acid/base added
- **Contains:** Weak acid + its salt OR weak base + its salt
- **Example:** CH‚āÉCOOH/CH‚āÉCOONa (acetic acid/sodium acetate)

### **How Buffers Work:**
**When acid (H‚Āļ) added:**
- H‚Āļ + CH‚āÉCOO‚ĀĽ ‚Üí CH‚āÉCOOH (salt uses up added H‚Āļ)

**When base (OH‚ĀĽ) added:**
- OH‚ĀĽ + CH‚āÉCOOH ‚Üí CH‚āÉCOO‚ĀĽ + H‚āāO (acid uses up added OH‚ĀĽ)

### **Henderson-Hasselbalch Equation:**
**For acid buffer:** pH = pKa + log([salt]/[acid])
**For base buffer:** pOH = pKb + log([salt]/[base])

**Example:** Buffer with 0.1 M CH‚āÉCOOH and 0.1 M CH‚āÉCOONa, Ka = 1.8√ó10‚ĀĽ‚ĀĶ
pKa = -log(1.8√ó10‚ĀĽ‚ĀĶ) = 4.74
pH = 4.74 + log(0.1/0.1) = 4.74 + log(1) = 4.74 + 0 = 4.74

### **Biological Buffers:**
- **Blood:** H‚āāCO‚āÉ/HCO‚āÉ‚ĀĽ maintains pH ~7.4
- **Important** for enzyme function, oxygen transport

## **Chapter 8: Salt Hydrolysis**

### **What is Salt Hydrolysis?**
- **Salt reacts with water** to produce acidic/basic solution
- **Depends on** strength of parent acid/base

### **Four Types of Salts:**

#### **1. Strong acid + Strong base salt:**
- **Example:** NaCl (from HCl + NaOH)
- **pH = 7** (neutral)
- **No hydrolysis**

#### **2. Strong acid + Weak base salt:**
- **Example:** NH‚āĄCl (from HCl + NH‚āÉ)
- **pH < 7** (acidic)
- **Cation hydrolyzes:** NH‚āĄ‚Āļ + H‚āāO ‚áĆ NH‚āÉ + H‚āÉO‚Āļ

#### **3. Weak acid + Strong base salt:**
- **Example:** CH‚āÉCOONa (from CH‚āÉCOOH + NaOH)
- **pH > 7** (basic)
- **Anion hydrolyzes:** CH‚āÉCOO‚ĀĽ + H‚āāO ‚áĆ CH‚āÉCOOH + OH‚ĀĽ

#### **4. Weak acid + Weak base salt:**
- **Example:** CH‚āÉCOONH‚āĄ
- **pH depends on** relative Ka and Kb
- **Both ions hydrolyze**

### **Memory Trick:**
```
Strong parent dominates:
- Strong acid parent ‚Üí acidic salt
- Strong base parent ‚Üí basic salt
- Both strong ‚Üí neutral salt
```

## **Chapter 9: Common Exam Questions**

### **Question Type 1: Multiple Choice**
**Example:** Which has pH > 7?
A) HCl  
B) NaOH  
C) CH‚āÉCOOH  
D) NH‚āĄCl  
**Answer:** B

### **Question Type 2: pH Calculation**
**Example:** Calculate pH of 0.01 M HCl.
**Solution:** [H‚Āļ] = 0.01 M, pH = -log(0.01) = 2

### **Question Type 3: Titration Calculation**
**Example:** 20.0 mL 0.1 M NaOH titrated with 0.2 M HCl. Calculate volume HCl needed.
**Solution:** M‚āĀV‚āĀ = M‚āāV‚āā ‚Üí 0.1√ó20.0 = 0.2√óV‚āā ‚Üí V‚āā = 2.0/0.2 = 10.0 mL

### **Question Type 4: Buffer Calculation**
**Example:** Buffer contains 0.2 M HA and 0.1 M A‚ĀĽ. Ka = 1√ó10‚ĀĽ‚ĀĶ. Calculate pH.
**Solution:** pH = pKa + log([A‚ĀĽ]/[HA]) = 5 + log(0.1/0.2) = 5 + log(0.5) = 5 - 0.3 = 4.7

### **Question Type 5: Explanation**
**Example:** Explain why NH‚āĄCl solution is acidic.
**Answer:** NH‚āĄ‚Āļ hydrolyzes: NH‚āĄ‚Āļ + H‚āāO ‚áĆ NH‚āÉ + H‚āÉO‚Āļ, producing H‚āÉO‚Āļ ions.
