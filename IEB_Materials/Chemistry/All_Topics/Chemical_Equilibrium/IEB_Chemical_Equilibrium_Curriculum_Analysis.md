## **Part 5: Key Concepts Simplified**

### **1. Dynamic Equilibrium:**
- **Forward rate = Reverse rate** (not stopped!)
- **Concentrations constant** (but not equal)
- **Macroscopic properties constant** (color, pressure, etc.)
- **Microscopic activity continues** (particles still reacting)

### **2. Equilibrium Constant Kc:**
- **Ratio** of product concentrations to reactant concentrations
- **Constant at given temperature** (changes with temperature only)
- **Large Kc** → Products favored (equilibrium lies to right)
- **Small Kc** → Reactants favored (equilibrium lies to left)

### **3. Le Chatelier's Principle:**
- **"If you disturb a system at equilibrium, it will shift to counteract the disturbance"**
- **Analogy:** Crowded room - if more people enter (disturbance), some leave (shift) to reduce crowding

### **4. Factors Affecting Equilibrium:**

#### **Concentration Changes:**
- **Add reactant** → Shift to products (use up added reactant)
- **Remove product** → Shift to products (make more product)

#### **Pressure Changes (gases only):**
- **Increase pressure** → Shift to side with fewer gas molecules
- **Decrease pressure** → Shift to side with more gas molecules

#### **Temperature Changes:**
- **Increase temperature** → Shift in endothermic direction
- **Decrease temperature** → Shift in exothermic direction

### **5. Reaction Quotient Q:**
- **Same formula as Kc** but using current concentrations
- **Q < K** → Shift right (toward products)
- **Q > K** → Shift left (toward reactants)
- **Q = K** → At equilibrium

## **Part 6: Common Student Misconceptions**

### **Misconception 1:** "Equilibrium means everything has stopped"
- **Correction:** Dynamic equilibrium - reactions continue at equal rates

### **Misconception 2:** "At equilibrium, concentrations are equal"
- **Correction:** Rates are equal, concentrations are constant but usually not equal

### **Misconception 3:** "Catalysts affect equilibrium position"
- **Correction:** Catalysts speed up both forward and reverse equally - no effect on position

### **Misconception 4:** "Changing pressure always affects equilibrium"
- **Correction:** Only if Δn(gas) ≠ 0 (different number of gas molecules on each side)

### **Misconception 5:** "Kc has units"
- **Correction:** Kc is unitless (activities used, but concentrations give units that cancel)

## **Part 8: Required Practical Investigations**

### **Essential Experiments:**

#### **1. Cobalt Chloride Equilibrium:**
- **System:** CoCl₄²⁻(blue) + 6H₂O ⇌ Co(H₂O)₆²⁺(pink) + 4Cl⁻
- **Disturbances:** Add water, add HCl, heat, cool
- **Skills:** Observation, explanation using Le Chatelier

#### **2. Nitrogen Dioxide Equilibrium:**
- **System:** 2NO₂(brown) ⇌ N₂O₄(colorless)
- **Disturbances:** Change pressure (syringe), change temperature
- **Skills:** Gas equilibrium observations, pressure effects

#### **3. Esterification Equilibrium:**
- **System:** Acid + alcohol ⇌ ester + water
- **Observations:** Smell changes, equilibrium shifting
- **Skills:** Organic equilibrium, real-world connection

#### **4. Iron(III) Thiocyanate Equilibrium:**
- **System:** Fe³⁺ + SCN⁻ ⇌ FeSCN²⁺ (blood red)
- **Disturbances:** Add ions, dilute, temperature
- **Skills:** Colorimetric analysis, quantitative observations

## **Part 9: IEB Exam Question Types**

### **Type A: Multiple Choice (Knowledge)**
**Example:** At dynamic equilibrium:
A) Forward reaction stops  
B) Reverse reaction stops  
C) Concentrations become equal  
D) Forward rate = reverse rate  
**Answer:** D

### **Type B: Kc Calculations (Application)**
**Example:** For N₂ + 3H₂ ⇌ 2NH₃, equilibrium concentrations: [N₂]=0.5M, [H₂]=1.5M, [NH₃]=2.0M. Calculate Kc.
**Solution:** Kc = [NH₃]²/([N₂][H₂]³) = (2.0)²/(0.5×(1.5)³) = 4/(0.5×3.375) = 4/1.6875 = 2.37

### **Type C: Le Chatelier Predictions (Analysis)**
**Example:** For 2SO₂ + O₂ ⇌ 2SO₃ (ΔH = -197 kJ), predict effect of increasing temperature.
**Answer:** Shift left (toward reactants, endothermic direction)

### **Type D: ICE Table Problems (Application)**
**Example:** Given initial concentrations and Kc, calculate equilibrium concentrations.
**Solution:** Set up ICE table, solve quadratic equation.

### **Type E: Industrial Applications (Evaluation)**
**Example:** Explain why the Haber process uses high pressure.
**Answer:** Shift equilibrium toward NH₃ (fewer gas molecules on product side), increases yield.

## **Part 12: Sample Lesson Plan (IEB Grade 12)**

### **Lesson: ICE Table Calculations**
**Duration:** 1 hour

#### **Learning Objectives:**
- Set up ICE tables for equilibrium problems
- Use ICE tables to calculate equilibrium concentrations
- Solve quadratic equations from equilibrium expressions

#### **Resources:**
- Worksheets with ICE table templates
- Calculators
- Whiteboard for worked examples

#### **Lesson Structure:**

**Starter (5 min):**
- Quick review: What is Kc?
- Simple Kc calculation example

**Direct Instruction (15 min):**
- Explain ICE table: Initial, Change, Equilibrium
- Demonstrate with simple example
- Show how to derive change expressions

**Guided Practice (20 min):**
- Work through example on board
- Step-by-step: ICE table → equilibrium expression → solve
- Common mistakes highlighted

**Independent Practice (15 min):**
- Worksheet with 3 ICE table problems
- Include both simple and requiring quadratic
- Peer checking encouraged

**Plenary (5 min):**
- Key takeaway: ICE tables organize equilibrium problems
- Exit ticket: "What does 'C' stand for in ICE table?"

#### **Homework:**
- Textbook problems on ICE tables
- Create own ICE table problem

#### **Assessment:**
- Formative: Worksheet completion
- Summative: ICE table question in next test

## **Conclusion**

Chemical equilibrium provides a crucial understanding of how chemical systems behave under different conditions. By connecting microscopic particle behavior with macroscopic observations, students develop both conceptual understanding and practical problem-solving skills.

The key to successful IEB teaching is balancing theoretical understanding with practical applications, particularly those relevant to South African industry and environment.

**Remember:** The goal is not just to calculate Kc values, but to understand how equilibrium principles govern chemical systems in industry, biology, and the environment.
