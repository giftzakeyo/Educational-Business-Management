## Worked Example 1: Ohm's Law

**1.1** Calculate the resistance:
```
R = V/I
R = 12/2.5
R = 4.8 Ω ✓
```

**1.2** New current:
```
I = V/R
I = 12/2
I = 6 A ✓
```

**1.3** Explanation:
Ohm's Law states I ∝ V (at constant temperature). For the same voltage, decreasing resistance allows more current to flow. Lower resistance offers less opposition to electron flow. ✓

## Worked Example 2: Internal Resistance

**2.1** Definition of emf:
The total energy supplied per coulomb of charge by the cell/battery, measured in volts when no current flows (open circuit). ✓

**2.2** Current:
```
ε = I(R + r)
24 = I(5.2 + 0.8)
24 = 6I
I = 4 A ✓
```

**2.3** Terminal voltage:
```
V = ε - Ir
V = 24 - (4 × 0.8)
V = 24 - 3.2
V = 20.8 V ✓
```
OR: V = IR = 4 × 5.2 = 20.8 V ✓

**2.4** Change in terminal voltage:
```
1/R_p = 1/5.2 + 1/5.2 = 2/5.2
R_p = 2.6 Ω
ε = I(2.6 + 0.8)
24 = 3.4I
I = 7.06 A
V = ε - Ir = 24 - (7.06 × 0.8)
V = 24 - 5.65
V = 18.35 V ✓
```
**Explanation:** The terminal voltage DECREASES from 20.8 V to 18.35 V. Adding a parallel resistor decreases total external resistance, which increases the current. The larger current causes a larger voltage drop across the internal resistance (Ir), leaving less voltage available at the terminals. ✓

## Worked Example 3: Combination Circuits

**3.1** Current through 6 Ω:
```
In parallel: V_same = 18 V
I = V/R = 18/6 = 3 A ✓
```

**3.2** Current through 3 Ω:
```
I = V/R = 18/3 = 6 A ✓
```

**3.3** Value of resistor R:
```
I_total = I_6Ω + I_3Ω + I_R
12 = 3 + 6 + I_R
I_R = 3 A
R = V/I = 18/3 = 6 Ω ✓
```

## Worked Example 4: Power & Energy

**4.1** Current drawn:
```
P = VI
I = P/V = 2200/230
I = 9.57 A ✓
```

**4.2** Resistance:
```
P = V²/R
R = V²/P = 230²/2200
R = 52900/2200
R = 24.05 Ω ✓
```

**4.3** Energy in kWh:
```
E = P × t
E = 2.2 kW × (15/60) h
E = 2.2 × 0.25
E = 0.55 kWh ✓
```

**4.4** Cost:
```
Cost = 0.55 × R2.15
Cost = R1.18 ✓
```

## Practice Activity 1: Internal Resistance (Memo)

**P1.1** Current:
```
ε = I(R + r)
1.5 = I(2.8 + 0.2)
1.5 = 3I
I = 0.5 A ✓
```

**P1.2** Terminal voltage:
```
V = IR = 0.5 × 2.8 = 1.4 V ✓
```
OR: V = ε - Ir = 1.5 - (0.5 × 0.2) = 1.5 - 0.1 = 1.4 V ✓

**P1.3** New terminal voltage:
```
1/R_p = 1/2.8 + 1/2.8 = 2/2.8
R_p = 1.4 Ω
ε = I(1.4 + 0.2)
1.5 = 1.6I
I = 0.9375 A
V = ε - Ir = 1.5 - (0.9375 × 0.2)
V = 1.5 - 0.1875
V = 1.3125 V ✓
```

**P1.4** Comparison:
Terminal voltage dropped from 1.4 V to 1.31 V. The parallel combination reduced total external resistance, increasing current. The larger current produces a larger voltage drop across the internal resistance (Ir), reducing terminal voltage. ✓

## Practice Activity 2: Series-Parallel Circuit (Memo)

**P2.1** Total resistance:
```
Parallel: 1/R_p = 1/4 + 1/6 = 3/12 + 2/12 = 5/12
R_p = 12/5 = 2.4 Ω
Total: R_T = 3 + 2.4 = 5.4 Ω ✓
```

**P2.2** Total current:
```
I = V/R = 12/5.4 = 2.22 A ✓
```

**P2.3** Voltage across 3 Ω:
```
V_3Ω = I × R = 2.22 × 3 = 6.67 V ✓
```

**P2.4** Current through 4 Ω:
```
Voltage across parallel = 12 - 6.67 = 5.33 V
I_4Ω = 5.33/4 = 1.33 A ✓
```

**P2.5** Power in 6 Ω:
```
I_6Ω = 5.33/6 = 0.889 A
P = I²R = (0.889)² × 6 = 0.79 × 6 = 4.74 W ✓
```
OR: P = V²/R = (5.33)²/6 = 28.4/6 = 4.73 W ✓

## Practice Activity 3: V vs I Graph (Memo)

**P3.1** Graph:
Plot points: (0, 9.0), (0.5, 8.5), (1.0, 8.0), (1.5, 7.5), (2.0, 7.0), (2.5, 6.5)
Line of best fit through all points. ✓
(Axes labelled: Voltage (V) on y-axis, Current (A) on x-axis)

**P3.2** Emf:
```
y-intercept = 9.0 V (voltage when I = 0)
ε = 9.0 V ✓
```

**P3.3** Gradient:
```
gradient = ΔV/ΔI = (6.5 - 9.0)/(2.5 - 0.0)
gradient = -2.5/2.5
gradient = -1.0 ✓
```

**P3.4** Physical meaning:
The gradient represents the **negative of the internal resistance**, i.e., gradient = -r. ✓
Therefore r = 1.0 Ω ✓

**P3.5** Equation:
```
V = ε - Ir
V = 9.0 - 1.0I
V = 9 - I ✓
```

**P3.6** Short-circuit current:
```
At short circuit, V = 0
0 = 9 - I
I = 9 A ✓
```
