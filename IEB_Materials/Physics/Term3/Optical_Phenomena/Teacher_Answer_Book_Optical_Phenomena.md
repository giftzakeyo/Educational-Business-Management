## Worked Example 1: Photon Energy

**1.1** Convert to metres:
```
500 nm = 500 × 10⁻⁹ m = 5.0 × 10⁻⁷ m ✓
```

**1.2** Photon energy:
```
E = hc/λ
E = (6.63 × 10⁻³⁴ × 3 × 10⁸) / (5.0 × 10⁻⁷)
E = (1.989 × 10⁻²⁵) / (5.0 × 10⁻⁷)
E = 3.98 × 10⁻¹⁹ J ✓
```

**1.3** Convert to eV:
```
E = 3.98 × 10⁻¹⁹ / 1.6 × 10⁻¹⁹
E = 2.49 eV ✓
```

**1.4** Type of EM radiation:
Visible light (blue-green region, since 500 nm is in the visible spectrum) ✓

## Worked Example 2: Photoelectric Effect

**2.1** Work function definition:
The minimum energy required to eject an electron from the surface of a metal. ✓

**2.2** Threshold frequency:
```
W₀ = hf₀
f₀ = W₀ / h
f₀ = (3.2 × 10⁻¹⁹) / (6.63 × 10⁻³⁴)
f₀ = 4.83 × 10¹⁴ Hz ✓
```

**2.3** Will electrons be ejected?
```
Incident frequency = 1.2 × 10¹⁵ Hz
Threshold frequency = 4.83 × 10¹⁴ Hz
Since f_incident > f₀, YES electrons will be ejected. ✓
Energy calculation: E = hf = 6.63×10⁻³⁴ × 1.2×10¹⁵ = 7.96×10⁻¹⁹ J
Since 7.96×10⁻¹⁹ > 3.2×10⁻¹⁹ (W₀), electrons are ejected ✓
```

**2.4** Maximum kinetic energy:
```
E_k(max) = hf - W₀
E_k(max) = (6.63×10⁻³⁴ × 1.2×10¹⁵) - 3.2×10⁻¹⁹
E_k(max) = 7.96×10⁻¹⁹ - 3.2×10⁻¹⁹
E_k(max) = 4.76 × 10⁻¹⁹ J ✓
```

**2.5** Maximum speed:
```
E_k = ½mv²
4.76 × 10⁻¹⁹ = ½ × (9.11×10⁻³¹) × v²
v² = (2 × 4.76×10⁻¹⁹) / (9.11×10⁻³¹)
v² = 1.045 × 10¹²
v = 1.02 × 10⁶ m·s⁻¹ ✓
```

## Worked Example 3: Photoelectric Effect Graph

**3.1** Graph:
Plot points: (4.0, 0), (5.0, 0.7), (6.0, 2.0), (7.0, 3.3), (8.0, 4.6), (9.0, 5.9)
Axes: E_k (×10⁻¹⁹ J) on y-axis, f (×10¹⁴ Hz) on x-axis
Line of best fit through all points ✓

**3.2** Threshold frequency:
The x-intercept (where E_k = 0):
```
f₀ = 4.0 × 10¹⁴ Hz ✓
```

**3.3** Gradient:
```
gradient = ΔE_k / Δf
gradient = (5.9 - 0.0) × 10⁻¹⁹ / (9.0 - 4.0) × 10¹⁴
gradient = 5.9 × 10⁻¹⁹ / 5.0 × 10¹⁴
gradient = 1.18 × 10⁻³³ J·s
gradient ≈ 6.63 × 10⁻³⁴ J·s ✓
```
(Make sure students get close to Planck's constant)

**3.4** What gradient represents:
The gradient represents **Planck's constant (h)**. This is because E_k = hf - W₀ is in the form y = mx + c, where the gradient = h. ✓

**3.5** Work function:
```
W₀ = hf₀
W₀ = (6.63 × 10⁻³⁴)(4.0 × 10¹⁴)
W₀ = 2.65 × 10⁻¹⁹ J ✓
```
OR: y-intercept (extrapolate back) = -W₀ = -2.65 × 10⁻¹⁹ J

**3.6** Einstein's equation:
```
E_k = hf - W₀
E_k = (6.63×10⁻³⁴)f - 2.65×10⁻¹⁹ ✓
```

## Worked Example 4: Spectra

**4.1** Emission vs absorption spectrum:
- **Emission spectrum:** Bright coloured lines on a dark background, produced when excited electrons fall to lower energy levels, emitting photons of specific wavelengths ✓
- **Absorption spectrum:** Dark lines on a continuous spectrum, produced when white light passes through a cool gas and electrons absorb specific wavelengths to jump to higher energy levels ✓

**4.2** Different line spectra:
Each element has a unique atomic structure with different energy levels. The difference between energy levels determines the wavelengths of light absorbed/emitted, creating a unique "fingerprint" for each element. ✓

**4.3** Evidence for discrete energy levels:
The fact that only specific wavelengths (discrete lines) appear in emission/absorption spectra shows that electrons can only exist at specific energy levels. If energy levels were continuous, a continuous spectrum would be produced. The gaps between lines represent the gaps between energy levels. ✓

**4.4** Dual nature of light:
Light behaves as both a **wave** and a **particle**:
- **Wave nature:** Demonstrated by diffraction and interference (Thomas Young's double-slit experiment, Huygens) ✓
- **Particle nature:** Demonstrated by the photoelectric effect (Einstein - photons) ✓

## Practice Activity 1: Photon Energy (Memo)

**P1.1** Energy of blue light photon:
```
E = hf = (6.63×10⁻³⁴)(6.5×10¹⁴)
E = 4.31 × 10⁻¹⁹ J ✓
```

**P1.2** Wavelength:
```
E = hc/λ
λ = hc/E
λ = (6.63×10⁻³⁴ × 3×10⁸) / (4.5×10⁻¹⁹)
λ = 4.42 × 10⁻⁷ m
λ = 442 nm ✓
```

**P1.3** Compare red vs violet:
```
Red: E = hc/λ = (6.63×10⁻³⁴ × 3×10⁸) / (700×10⁻⁹) = 2.84×10⁻¹⁹ J
Violet: E = hc/λ = (6.63×10⁻³⁴ × 3×10⁸) / (400×10⁻⁹) = 4.97×10⁻¹⁹ J
Violet has MORE energy (shorter wavelength = higher frequency = more energy) ✓
```

**P1.4** Number of photons per second:
```
E_per_photon = hc/λ = (6.63×10⁻³⁴ × 3×10⁸) / (550×10⁻⁹) = 3.62×10⁻¹⁹ J
Number = Total energy per second / Energy per photon
Number = 60 / (3.62×10⁻¹⁹)
Number = 1.66 × 10²⁰ photons/s ✓
```

## Practice Activity 2: Photoelectric Effect (Memo)

**P2.1** Work function in joules:
```
W₀ = 2.28 × 1.6×10⁻¹⁹ = 3.65 × 10⁻¹⁹ J ✓
```

**P2.2** Threshold frequency:
```
f₀ = W₀/h = 3.65×10⁻¹⁹ / 6.63×10⁻³⁴
f₀ = 5.51 × 10¹⁴ Hz ✓
```

**P2.3** Will emission occur?
```
Incident f = 8.0×10¹⁴ Hz > f₀ = 5.51×10¹⁴ Hz
YES, photoelectric emission will occur ✓
```

**P2.4** Maximum kinetic energy:
```
E_k = hf - W₀ = (6.63×10⁻³⁴ × 8.0×10¹⁴) - 3.65×10⁻¹⁹
E_k = 5.30×10⁻¹⁹ - 3.65×10⁻¹⁹
E_k = 1.65 × 10⁻¹⁹ J ✓
E_k in eV = 1.65×10⁻¹⁹ / 1.6×10⁻¹⁹ = 1.03 eV ✓
```

**P2.5** Effect of doubling intensity:
- **P2.5.1:** Kinetic energy **unchanged** (KE depends on frequency, not intensity) ✓
- **P2.5.2:** Number of electrons ejected per second **increases** (more photons means more electrons) ✓

## Practice Activity 3: Einstein's Equation (Memo)

**P3.1** Frequency of incident light:
```
c = fλ
f = c/λ = 3×10⁸ / (400×10⁻⁹)
f = 7.5 × 10¹⁴ Hz ✓
```

**P3.2** Photon energy:
```
E = hf = 6.63×10⁻³⁴ × 7.5×10¹⁴
E = 4.97 × 10⁻¹⁹ J ✓
```

**P3.3** Will emission occur?
```
W₀ = hf₀ = 6.63×10⁻³⁴ × 5.0×10¹⁴ = 3.32 × 10⁻¹⁹ J
E_photon (4.97×10⁻¹⁹) > W₀ (3.32×10⁻¹⁹)
YES, emission will occur ✓
```

**P3.4** Maximum KE:
```
E_k = hf - W₀ = 4.97×10⁻¹⁹ - 3.32×10⁻¹⁹
E_k = 1.65 × 10⁻¹⁹ J ✓
```

**P3.5** Effect of increased intensity (same wavelength):
- **Number of photoelectrons:** INCREASES (more photons incident) ✓
- **Kinetic energy:** UNCHANGED (same frequency = same photon energy, so E_k = hf - W₀ unchanged) ✓
