# Electrodynamics - Grade 12 IEB Physics (Term 3)

## IEB Curriculum Alignment

### Topics Covered:
1. **Electric Motors** — DC motor, split-ring commutator, Fleming's Left Hand Rule
2. **Electric Generators** — AC generator (alternator), slip rings, Fleming's Right Hand Rule
3. **AC vs DC** — Differences, graphs, applications
4. **RMS values** — Root mean square for AC voltage and current

### Key Equations:
- For a generator: ε = -NΔΦ/Δt (Faraday's Law - qualitative)
- RMS voltage: V_rms = V_max/√2
- RMS current: I_rms = I_max/√2
- P_avg = V_rms × I_rms

### IEB Exam Weighting:
- Usually **10-18 marks** in Paper 1
- High-frequency: DC motor operation, AC generator graphs, RMS calculations
- Often includes a diagram labelling question
