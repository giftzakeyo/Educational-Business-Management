# Lab Activity: Verifying F=ma

**Title:** Force, Mass, and Acceleration Experiment  
**Objective:** To verify Newton's Second Law (F=ma) by measuring how acceleration changes with force and mass.  
**Duration:** 45-60 minutes  
**Group Size:** 2-3 students

---

## **Materials Needed Per Group:**
- Dynamics cart or smooth-rolling toy car
- String (1 meter)
- Mass set (50g, 100g, 200g masses)
- Pulley (clamped to table edge)
- Stopwatch or smartphone timer
- Meter stick or measuring tape
- Spring scale (optional, for force measurement)
- Graph paper
- Calculator

## **Safety Precautions:**
- Keep work area clear
- Secure pulley firmly
- Don't let masses fall on feet
- Wear closed-toe shoes

---

## **Part A: Acceleration vs. Force (Constant Mass)**

**Hypothesis:** If we increase the force while keeping mass constant, acceleration will ________________.

**Procedure:**
1. Set up the apparatus as shown:
   ```
   [Cart] ---- [String] ---- [Pulley] ---- [Hanging Mass]
           (on table)                   (falling vertically)
   ```

2. Use a cart mass of approximately 500g (measure exact mass: ______ kg).

3. **Trial 1:** Use 50g hanging mass.
   - Measure distance cart travels: d = 1.0 meter
   - Release cart from rest
   - Time how long it takes to travel 1.0 meter: t = ______ s
   - Calculate acceleration: a = 2d/t² = 2(1.0)/t² = ______ m/s²
   - Calculate force: F = m_hanging × g = (______ kg) × 9.8 m/s² = ______ N

4. **Repeat** with 100g and 200g hanging masses.

**Data Table A:**

| Trial | Hanging Mass (kg) | Force, F (N) | Time, t (s) | Acceleration, a (m/s²) |
|-------|-------------------|--------------|-------------|------------------------|
| 1     | 0.050             |              |             |                        |
| 2     | 0.100             |              |             |                        |
| 3     | 0.200             |              |             |                        |

**Calculations for Trial 1 (show work):**
a = 2d/t² = 2(______)/(______)² = ______ m/s²  
F = m × g = ______ × 9.8 = ______ N

## **Part B: Acceleration vs. Mass (Constant Force)**

**Hypothesis:** If we increase the mass while keeping force constant, acceleration will ________________.

**Procedure:**
1. Use a constant hanging mass of 100g (F = 0.100 kg × 9.8 = 0.98 N).

2. **Trial 1:** Cart only (no extra mass).
   - Measure cart mass: m_total = ______ kg
   - Time over 1.0 meter: t = ______ s
   - Calculate acceleration: a = ______ m/s²

3. **Trial 2:** Add 200g to cart.
   - m_total = cart mass + 0.200 kg = ______ kg
   - Time: t = ______ s
   - a = ______ m/s²

4. **Trial 3:** Add 500g to cart.
   - m_total = ______ kg
   - Time: t = ______ s
   - a = ______ m/s²
